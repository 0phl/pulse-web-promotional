# PULSE - Community App Promotional Website

A promotional landing page for **PULSE** (Public Updates, Local Services, and Engagement) - a mobile app that connects communities through notices, marketplace, volunteering, and reports.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Hero section with app preview
- Feature highlights with phone mockups
- Download section with APK link
- Fully responsive design
